# NeuroCraft Studio

Образовательная веб-платформа для изучения нейронных сетей: визуальный
конструктор архитектур, генерация кода, обратная визуализация кода в схему,
обучение моделей прямо в браузере и база знаний.

## Возможности

- **Редактор архитектур** — сборка сетей из слоёв с живой D3-схемой и генерацией
  кода под три фреймворка (Keras / PyTorch / TensorFlow.js). Поддерживаются пять
  семейств:
  - перцептрон (полносвязная сеть);
  - свёрточная сеть (CNN) — карты признаков, пулинг, flatten;
  - рекуррентная сеть (LSTM / GRU / SimpleRNN);
  - трансформер (энкодер с Multi-Head Attention);
  - GAN (генератор + дискриминатор).
    Доступны слои регуляризации (Dropout, BatchNorm), вставка/перемещение слоёв,
    настройка цветов схемы, сплошная заливка, экспорт схемы в PNG, подсказки и
    мягкая валидация архитектуры.
- **Визуализация кода → архитектуры** — вставьте код модели, и платформа
  распознает слои и нарисует схему в том же стиле, что и редактор (CNN — карты
  признаков, RNN/трансформер — энкодер с петлёй состояния, GAN — состязательная
  схема, перцептрон — нейроны). Разбор эвристический и рассчитан прежде всего на
  декларативное описание слоёв (Keras, TensorFlow.js); классовый код PyTorch
  распознаётся частично.
- **Песочница** — обучение небольших сетей на 2D-данных прямо в браузере
  (TensorFlow.js, ленивая загрузка), кривые потерь/точности и граница решения в
  стиле TensorFlow Playground.
- **База знаний** — статьи по теории с глоссарием и перекрёстными ссылками.
- **Личный кабинет** — сохранение и загрузка собранных архитектур.

## Технологии

- **Vite 6** + **React 18** + **TypeScript**
- **D3** — все схемы архитектур
- **@monaco-editor/react** — редактор/просмотр кода
- **@tensorflow/tfjs** — обучение в песочнице (отдельный чанк, ленивая загрузка)
- **@tanstack/react-query** — работа с API
- **react-router-dom** — маршрутизация
- **react-hook-form** + **zod** — формы и валидация
- **SCSS** и **CSS Modules** — стили

## Запуск

```bash
npm install
npm run dev       # дев-сервер (http://localhost:5173)
npm run build     # продакшен-сборка (tsc + vite build)
npm run preview   # предпросмотр собранной версии
npm run lint      # проверка ESLint
```

## Структура проекта

```
src/
  pages/
    Home/                     — главная (hero, возможности, карусель)
    CodeGeneration/           — редактор архитектур (5 семейств) + общие
                                D3-рендеры (drawCnnDiagram / drawSequenceDiagram /
                                drawGanDiagram), генерация кода
    VisualizationArсhitecture/— визуализация кода в схему
    Playground/               — обучение в браузере (TF.js)
    KnowledgeBase/            — база знаний
    Login / Registration / Account
  api/
    mock/                     — локальные моки: codegen.ts (архитектура → код),
                                parseCode.ts (код → архитектура), utils.ts
    Network.ts / Code.ts      — типы и обёртки вызовов
  auth/                       — мок-авторизация и сохранённые проекты (localStorage)
  data/                       — глоссарий, шаблоны
  components/                 — Header, Footer, Button, GlossaryTerm и др.
```
